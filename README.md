PROJECT AGILE
SmartWaste (Sistem Pengelolaan Sampah Digital)
Deskripsi Project
SmartWaste adalah aplikasi berbasis web yang bertujuan membantu masyarakat dalam mengelola 
sampah secara lebih efektif dan bernilai ekonomi. Sistem ini memungkinkan pengguna untuk 
mencatat sampah, menjadwalkan penjemputan, serta mendapatkan reward dari sampah yang 
dikumpulkan.
Project ini dibuat untuk mengatasi masalah kurangnya kesadaran masyarakat dalam pengelolaan 
sampah serta memanfaatkan sampah yang masih memiliki nilai guna. Dengan adanya platform ini, 
diharapkan tercipta lingkungan yang lebih bersih serta sistem ekonomi berbasis daur ulang.
15 FITUR PENTING
A. Sistem Pengguna
1. Registrasi akun menggunakan email 
2. Login pengguna 
3. Manajemen profil pengguna 
B. Manajemen Sampah
4. Input jenis sampah (plastik, kertas, organik, dll) 
5. Input berat sampah 
6. Upload foto sampah 
7. Estimasi nilai sampah otomatis 
C. Sistem Penjemputan
8. Request penjemputan sampah 
9. Penjadwalan pickup 
10. Tracking status penjemputan 
11. Riwayat penjemputan 
D. Sistem Reward
12. Pemberian poin berdasarkan sampah 
13. Penukaran poin menjadi uang atau voucher 
E. Monitoring dan Informasi
14. Dashboard statistik pengguna 
15. Notifikasi sistem (status pickup dan poin)
